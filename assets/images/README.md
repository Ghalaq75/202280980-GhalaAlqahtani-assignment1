# Images Directory

Place your images here for the portfolio:

- Profile photo (recommended: 400x400px)
- Project screenshots (recommended: 800x600px)
- Any other visual assets

## Recommended Format
- **Profile**: .jpg or .png, optimized for web
- **Projects**: .jpg or .png, compressed for faster loading

## Adding Images

1. Add your images to this directory
2. Update the HTML to reference them:

### Profile Image
Replace the placeholder in the About section:
```html
<div class="about-image">
    <img src="assets/images/profile.jpg" alt="Ghala Alqahtani">
</div>
```

### Project Images
Replace placeholders in project cards:
```html
<div class="project-image">
    <img src="assets/images/project1.jpg" alt="Project Name">
</div>
```

## Image Optimization Tips

- Compress images before uploading (use tools like TinyPNG)
- Use appropriate dimensions (don't use 4000px image for 400px display)
- Consider using WebP format for better compression
- Add descriptive alt text for accessibility
